게임이 시작되고 적이 생성되면 적이 플레이어에게 이동 
적이 플레이어에게 닿았나 확인 안 닿았을 경우 적이 총알에 맞았는지 안맞았는지 판단
적이 총알에 맞았다면 크기와 hp가 감소하고 플레이어에게 이동 맞지 않았다면 계속 플레이어에게 이동
적의 체력이 0이 되었을 때 점수를 획득 그 후 플레이어어가 살아있는지 확인
적의 체력이 0인데 살아있다면 다시 적을 생성하고 플레이어가 죽었다면 지금까지 얻은 점수를 출력 후 게임종료

적이 플레이어에게 닿았다면 플레이어의 hp가 감소하고 적이 사라짐 그때 플레이어 체력이 0인지 아닌지 판단
0이라면 얻은 점수 출력과 함께 게임을 종료하고 0이 아니라면 감소 후 적을생성(계속진행)
